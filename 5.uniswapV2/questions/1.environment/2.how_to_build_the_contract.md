# Q2: 如何构建（编译）合约？

环境版本要求见 `questions/1.environment.md`。

## 步骤

1. 安装依赖（需先满足 Node 16.x、Yarn 1.x 等）：

   ```bash
   yarn
   ```

2. 编译合约（生成 `build/` 产物）：

   ```bash
   yarn compile
   ```

   该命令会执行 `precompile`（清理 `build/`），再调用 Waffle（`waffle .waffle.json`），使用项目内的 **solc 0.5.16**。

3. （可选）运行测试（会先编译再跑 Mocha）：

   ```bash
   yarn test
   ```
