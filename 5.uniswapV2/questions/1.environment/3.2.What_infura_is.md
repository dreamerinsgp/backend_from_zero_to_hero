What Infura is
Infura is a hosted service that runs Ethereum nodes (and other networks) for you and exposes them over the internet as HTTPS URLs (JSON-RPC “endpoints”).

Your wallet or script does not talk to the blockchain “directly” by magic—it needs a connection to a node that speaks the Ethereum JSON-RPC protocol. Infura is one provider that gives you that connection.

Why you might need it (or something like it)
To send transactions and read chain state from code (e.g. a deploy script using ethers), you need an RPC URL pointing to a node on the right network (e.g. Sepolia).
Running your own Ethereum node is heavy (sync, disk, maintenance). For development and testnets, people usually use a provider instead: Infura, Alchemy, Ankr, public RPCs, etc.
So you don’t have to use Infura specifically—you need some RPC endpoint. Infura is just a popular way to get one after you sign up and create a project.

Short summary
Concept	Meaning
Infura	A company that hosts blockchain nodes and gives you RPC URLs.
Why	Your scripts need a node to deploy contracts and call functions; Infura (or similar) supplies that without you running a node yourself.
