# Q1：如何查看池子当前的流动性？

在 Uniswap V2 里，“池子里的流动性”通常指：

- **`getReserves()`**：**token0 / token1** 在 Pair 合约里的储备量（恒定乘积里的 \(x\)、\(y\)）。  
- **`totalSupply()`**：**LP 代币**总发行量（Pair 继承 ERC20，LP 本身是一种 ERC20）。

本仓库提供只读脚本（**不需要私钥**，只要 **`RPC_URL`**）。

---

## 脚本用法

1. **`yarn compile`**（生成 **`build/UniswapV2Pair.json`**）。  
2. 在 **`.env`** 中设置：  
   - **`RPC_URL`** — 与 Pair 所在网络一致（如 Sepolia）。  
   - **`PAIR_ADDRESS`** — Pair 合约地址。  
3. 运行：

```bash
yarn check:liquidity
```

终端会打印 **`token0` / `token1` 地址**、**储备（raw 与按 decimals 格式化）**、**`blockTimestampLast`**、**LP `totalSupply`**。

示例（你文档中的 Pair）：

```env
PAIR_ADDRESS=0x108CeF10e86e7834660D97C3a008c29a2Fd241D7
```

---

## 说明

- 若尚未向池子转入代币并 **`mint`**，储备多为 **0**，LP **总供应**也可能只有首铸相关行为后的结果；见 `questions/6.add_liquidity.md`。  
- 若某代币 **`symbol()` 非标准**，脚本可能显示 `?`，但 **储备 raw 值**仍可用。
