# Q1：如何调用 `swap`？需要哪些前提？

本仓库 **`UniswapV2Pair.swap(amount0Out, amount1Out, to, data)`** 是底层兑换接口：先**乐观**把 **token0 / token1** 转给 **`to`**，再读 Pair 余额，用 **`amount0In` / `amount1In`**（相对储备的增量）校验 **恒定乘积 + 0.3% 手续费**（`UniswapV2: K`）。

---

## 机制概要

1. **输出**：`amount0Out`、`amount1Out` 至少一个 **> 0**，且分别 **小于** 当前储备（`INSUFFICIENT_LIQUIDITY`）。  
2. **收款地址**：**`to` 不能是 `token0` 或 `token1` 地址**（`INVALID_TO`），否则 revert。  
3. **输入**：合约**不会**从 `msg.sender` `transferFrom`；输入必须体现为 Pair **余额相对储备的增加**（通常在**同一笔交易**里先转入 Pair，或由 **flash swap** 回调里打回）。  
4. **Flash swap**：若 **`data.length > 0`**，会对 **`to`** 调用 **`uniswapV2Callee.uniswapV2Call`**，需在回调中把应付的代币转回 Pair。  
5. **注释**：与 `mint` 一样，生产环境通常由 **`v2-periphery` Router** 封装（路径、截止时间、滑点、WETH 等）。

---

## 前提条件清单

| 前提 | 说明 |
|------|------|
| **池子有储备** | **`getReserves()`** 两侧 **> 0**（先 **`createPair` + 加流动性**）。 |
| **输入代币** | 钱包里有足够 **`AMOUNT_IN`**，且能 **`transfer(pair, amount)`**。 |
| **方向与数量** | 明确是卖 **token0** 还是 **token1**，以及输入数量；输出由公式 **`getAmountOut`** 决定（含手续费）。 |
| **`to` 合法** | 接收输出的地址 **不是** 两个 ERC20 合约地址。 |
| **Gas** | 账户有足够原生币支付 **两笔交易**（见下）。 |

---

## 本仓库脚本（`yarn swap`）

脚本 **`scripts/swap-exact-in.js`** 实现 **精确输入（exact in）**：

1. 按 Uniswap V2 公式计算 **`getAmountOut`**；  
2. **`transfer`** 输入代币到 Pair；  
3. **`swap(amount0Out, amount1Out, SWAP_TO, 0x)`**（无 flash 回调）。

**环境变量：**

| 变量 | 说明 |
|------|------|
| **`RPC_URL`**、**`PRIVATE_KEY`** | 与 `add:liquidity` 相同。 |
| **`PAIR_ADDRESS`** | Pair 地址。 |
| **`SWAP_IN_TOKEN`** | **`0`** = 用 **token0** 换 **token1**；**`1`** = 用 **token1** 换 **token0**。 |
| **`AMOUNT_IN_WEI`** | 输入数量（wei 整数字符串）。 |
| **`SWAP_TO`**（可选） | 输出接收地址（默认 caller）。 |
| **`MIN_OUT_WEI`**（可选） | 若计算出的输出 **小于** 此值则 **revert**（简单滑点保护）。 |

```bash
yarn compile
yarn swap
```

**注意：** 脚本是 **两笔链上交易**（先转后 `swap`），与 Router **单笔原子**不同；主网更推荐 periphery。**`MIN_OUT_WEI`** 建议在生产或大额时使用，避免被抢跑或池状态变化坑到。

---

## 小结

- **前提**：有流动性的 Pair、钱包有足够 **输入代币** 与 **gas**、**`to` 不是 token0/token1**。  
- **脚本**：**`SWAP_IN_TOKEN`** + **`AMOUNT_IN_WEI`**，可选 **`SWAP_TO`** / **`MIN_OUT_WEI`**，执行 **`yarn swap`**。
