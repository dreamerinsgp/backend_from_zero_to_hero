 表创建基础

- `CREATE TABLE` 语法
- 字段定义：字段名、数据类型、约束
- `NOT NULL` 约束
- `DEFAULT` 默认值设置
- `COMMENT` 字段注释
- `AUTO_INCREMENT` 自增主键

**项目示例**：

```sql
CREATE TABLE `block` (
    `id` bigint NOT NULL AUTO_INCREMENT,
    `slot` bigint NOT NULL DEFAULT '0' COMMENT 'slot',
    `status` tinyint NOT NULL DEFAULT '0' COMMENT '1 processed, 2 failed',
    PRIMARY KEY (`id`)
)
```
