内容：
1.规则

1）leftmost prefix of the index （最左匹配原则）
Match the full value： Cuba Allen who was born on 1960-01-01
select * from People where last_name = "Allen" and first_name = "Cuba" and bod = "1960-01-01"

Match a leftmost prefix: Allen
select * from People where last_name = "Allen"

Match a column prefix
select * from People where last_name = "J"

Match a range of values
select * from People where last_name Between "Allen" and "Barrymore"

Match one part exactly and match a range on another part
select * from People where last_name = "Allen" and first_name = "K"

2) 字段不会命中索引的场景
a: 没有遵寻最左匹配原则
b: 跳过中间的索引字段：
select * from People where last_name = "Allen" and bod = "1960-01-01"
c: 范围查询后的字段
select * from People where last_name = "Allen" and first_name like "K" and bod = "1960-01-01"
