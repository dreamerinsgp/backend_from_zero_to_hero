这个项目是聚焦在后端快速入门上。 

如果大家想要高效的学习任何内容，有几个基本原则分享给大家：
1. 目标导向：任何跟你目标不想关,低相关度的内容，都是噪音，直接忽视 （你没有那么多的时间，去浪费在跟你目标不想管的事情上）
2. 知识边界：你要基于你的目标，来寻找合适的实际项目（如何定义合适？看你的目标是什么，比如说你想要转行，那么就看这些目标岗位的要求是什么？然后基于这些要求来检验项目的合适程度），来帮助你限定知识的边界。 从教程中学习，已经是过去时了。
3. 上手实践: "I hear and forget. I see, and I remember. I do, and I understand. " (你听过，你会忘记； 你看过你会记住； 你实践你会理解 )
